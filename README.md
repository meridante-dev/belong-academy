# Belong Academy

The learning academy of **Belong** — *giving shape to meaning*. Hospitality, seasonal
food, nature and community, taught the Belong way.

Built on the **Lumina** white-label learning-platform core (the same engine that powers
EdenRise Academy) — shared `core/`, with Belong as a thin instance in `brands/belong/`
(brand config + course catalog). Live via GitHub Pages.

- Brand: terracotta `#BE594D`, Source Serif Pro + Work Sans, the circle-over-bar symbol.
- Pillars → categories: Staying & Feeling · Growing & Tasting · To Nature · School.
